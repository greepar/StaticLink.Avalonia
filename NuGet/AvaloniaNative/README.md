# StaticLink.Avalonia.Native

Static `AvaloniaNative` libraries for macOS Avalonia NativeAOT publishing.

The package version matches the Avalonia version with a `.1` suffix. For example, use `StaticLink.Avalonia.Native` version `12.1.0.1` with Avalonia `12.1.0`.

```xml
<PackageReference Include="Avalonia" Version="12.1.0" />
<PackageReference Include="StaticLink.Avalonia.Native" Version="12.1.0.1" />
```

The package supports `osx-x64` and `osx-arm64` NativeAOT publishing.
